# Тесты для лабараторной работы №01

## Входные данные
Вещественные числа x1 y1 x2 y2 x3 y3

## Выходные данные
Число 1, 2 или 3

## Позитивные тесты:
- 01 - точка ниже прямой 
- 02 - точка выше прямой
- 03 - точка на прямой

## Негативные тесты:
- 01 - координаты точки - буквы
- 02 - точки лежащие на прямой совпадают

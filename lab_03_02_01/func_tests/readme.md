# Тесты для лабораторной работы №02

## Входные данные
Размер матрицы и сама матрица

## Выходные данные
Массив целых чисел

## Позитивные тесты
- 01 - матрица 2 на 2
- 02 - несколько подходящих элементов
- 03 - матрица из одинаковых элементов
- 04 - последний элемент матрицы подходящий

## Негативные тесты
- 01 - вводится только размер матрицы
- 02 - 11 строк
- 03 - 11 столбцов
- 04 - буквы вместо размеров матрицы
- 05 - 0 строк
- 06 - 0 столбцов
- 07 - некорректные символы среди элементов матрицы
